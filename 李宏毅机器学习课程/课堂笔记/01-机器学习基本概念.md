# 机器学习基本概念

机器学习 $\approx$ 寻找一个恰当的的函数。这个函数十分复杂，根本不可能用人力写出，所以用计算机寻找。

机器学习的任务可以分为以下几类：

- 回归(Regression)：如预测未来的房屋价格、股票走势、或者天气温度。
- 分类(Classification)：如垃圾邮件过滤(正常邮件/垃圾邮件)、疾病诊断、图像分类。
- 结构化学习(Structured Learning)：生成一些结构化的东西，如图片，文章。

寻找所需的那个恰当的函数的步骤：

- 写出带未知参数 $\boldsymbol{\theta}$ 的函数，这个函数就是模型，$\boldsymbol{\theta}$ 为未知参数组成的列向量。
- 定义以未知参数为自变量的损失函数 $\mathcal{L}$ ，用于衡量这组参数是否合理。常见的损失函数如：平均绝对误差(MAE)、均方差(MSE)、交叉熵(Cross Entropy)。
- 找到一组最好的参数$\boldsymbol{\theta}^*$，使得损失函数 $\mathcal{L}$ 达到最小。最初的参数 $\boldsymbol{\theta}^0$ ，往往是随机选择。参数更新时，需要先计算损失函数关于位置参数的梯度 $\nabla\mathcal{L}(\boldsymbol{\theta})$ 找出使损失函数变小得最快的方向，然后梯度乘以学习率 $\eta$ 来决定参数具体更新多少：
$$
\boldsymbol{\theta}^k = \boldsymbol{\theta}^{k-1} - \eta\cdot \left. \nabla\mathcal{L}\right|_{\boldsymbol{\theta}=\boldsymbol{\theta}^{k-1}}
$$


用来控制模型参数如何更新的参数就是超参数(就比如学习率 $\eta$ )，超参数通常由自己设置。

最简单的模型就是线性模型(Linear Model)，即 $\hat{y}=\omega \cdot x + b$ ，$x$ 为输入特征(features)，$\omega$ 为权重(weights)，$b$ 为偏置项(bias)。线性的模型往往有很大的缺陷，这种模型自身的缺陷称为 Model Bias 。

# 深度学习基本概念

为了解决linear model的固有缺陷，需要引入非线性因素，使用sigmoid函数就是其中一种方法，$\text{sigmoid}(x)=\frac{c}{1+e^{-x}}$ 。那原本的线性模型经过这样的改进后就变成了：
$$
\hat{y} = b + \vec{\boldsymbol{c}}^\top\text{sigmoid}(\vec{\boldsymbol{b}} + \boldsymbol{W}\vec{\boldsymbol{x}})
$$
$\vec{\boldsymbol{c}},\vec{\boldsymbol{b}},\vec{\boldsymbol{x}}$ 都是列向量，$\boldsymbol{W}$ 是特征的权重矩阵。

在实际操作中会有数据分批的操作，所以要区别以下名词：

- 批大小(Batch Size)：每次用于更新模型参数的数据量。如有 1000 条数据，设置 Batch Size 为 100，意味着模型每次看 100 条数据就更新一次参数。
- 轮次(Epoch)：当完整的数据集被完整地读取过一遍，就称为一个 Epoch。
- 更新次数(Updates)：$\text{Updates} = \lceil \frac{\text{Total Samples}}{\text{Batch Size}} \rceil$ 。

除了使用 $\text{sigmoid}(x)$ 之外，还有 $\text{ReLU}(x)=\max(0,x)$ 等，这些统称为激活函数(activation function)。

欠拟合：无法有效有效拟合训练数据，在训练数据和测试数据上误差都很大。

过拟合：在训练数据上表现很好，但在新的测试数据上误差却很大。
